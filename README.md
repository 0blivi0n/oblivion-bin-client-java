mercury-java-client
===================

##Introduction

  0blivi0n's binary API client for JAVA applications.
  
  
##Instalation

Maven dependency:
 
 ```xml
<dependency>
    <groupId>org.oblivion-cache</groupId>
    <artifactId>oblivion-bin-client</artifactId>
    <version>0.5.0</version>
</dependency>
 ```
